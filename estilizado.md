# Prompt de Diseño: Plataforma Inteligente de Autos Usados (Edición Luxury Glassmorphism)

[cite_start]Este prompt define la interfaz de usuario (UI) para una plataforma de compra y venta de vehículos, combinando una estética de lujo con tecnología de análisis por IA[cite: 8, 9].

## 1. Estética Visual y Colores
- **Paleta de Colores (Inspiración: ColorPaletteExample):**
    - **Primario:** Rojo deportivo y profundo (#D1001C) para botones de llamada a la acción (CTA) y acentos clave.
    - **Fondo:** Dark Mode elegante con negros profundos y gradientes oscuros para resaltar el brillo de los vehículos.
- **Efecto Glassmorphism (Inspiración: LuxuryCarDesignExample):**
    - Uso de contenedores con desenfoque de fondo (backdrop-filter: blur).
    - Bordes sutiles y semitransparentes que simulan vidrio esmerilado.
    - Tarjetas de información flotantes sobre imágenes de alta resolución.

## 2. Estructura de la Landing Page (Inspiración: LandingPageExample & LuxuryCarDesign)
- [cite_start]**Hero Section:** Imagen de un vehículo en primer plano con tipografía "Luxury" de gran tamaño y botones con bordes redondeados y efectos de brillo[cite: 206].
- **Sección de Beneficios:** Tres tarjetas con efecto Glassmorphism que describen:
    - [cite_start]**Análisis de Estado IA:** Evaluación del estado general del automóvil[cite: 10, 100].
    - [cite_start]**Detección de Daños:** Identificación visual de daños mediante IA[cite: 11, 99].
    - [cite_start]**Valor de Mercado:** Estimación aproximada del precio real[cite: 12, 101].
- [cite_start]**Sección de Marcas:** Fila de logotipos minimalistas en monocromo o dorado sutil para filtrar por marca[cite: 65, 77].
- **Preguntas Frecuentes (FAQ):** Acordeones integrados en contenedores de vidrio oscuro.

## 3. Sistema de Filtros y Búsqueda (Inspiración: CarFilterExample)
- **Filtro Lateral (Sidebar):**
    - [cite_start]**Histograma de Precios:** Gráfico de barras interactivo para seleccionar el rango de precio[cite: 79, 109].
    - [cite_start]**Selectores de Texto:** Campos para Marca, Modelo y Año[cite: 65, 66, 67, 80].
    - [cite_start]**Checkboxes de Tipo:** Opciones para Sedan, Coupe, SUV, Van, etc.[cite: 25, 47].
    - [cite_start]**Transmisión:** Botones segmentados para Manual o Automático[cite: 70].
- [cite_start]**Resultados de Búsqueda:** Grilla de vehículos con tarjetas que incluyen indicadores de IA (Excelente, Buen estado, Regular)[cite: 83, 103, 104, 105].

## 4. Funcionalidades Clave del Proyecto (Basado en PDF de Consignas)
- [cite_start]**Roles de Usuario:** Header dinámico con acceso para Comprador y Vendedor[cite: 59, 60, 61].
- [cite_start]**Detalle del Vehículo:** Galería de múltiples fotografías y visualización de características como kilometraje, combustible y ubicación[cite: 68, 69, 72, 86, 87].
- [cite_start]**Comparador Avanzado:** Tabla comparativa de atributos para múltiples vehículos seleccionados[cite: 50, 108, 241, 242].
- [cite_start]**Sección de Contacto:** Botón de contacto directo con el vendedor integrado en la ficha del auto[cite: 51, 115].

## 5. Especificaciones de Diseño
- **Tipografía:** Moderna, Sans-serif (ej. Inter o Poppins), con pesos variados para jerarquizar títulos de precios y modelos.
- **Componentes UI:** Inputs con estados focales en rojo, sombras suaves y transiciones fluidas para una sensación "Premium".
- [cite_start]**Responsive:** Adaptabilidad total desde dispositivos móviles hasta monitores ultra-wide[cite: 166].
