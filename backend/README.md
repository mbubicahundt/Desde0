# Backend — LuxAuto (NestJS)

Este backend es parte del proyecto LuxAuto. La guía completa de setup local y deploy está en el README raíz del repo:

- Ver: `README.md` en la raíz.

## Scripts

- `npm run start:dev` — desarrollo
- `npm run build` — compilar
- `npm run start:prod` — ejecutar dist (producción)

## Config (Railway)

Si deployás en Railway, este servicio incluye `nixpacks.toml` para compilar y arrancar en modo producción automáticamente.
